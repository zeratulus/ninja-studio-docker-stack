###Ninja Studio Docker Stack

Docker compose example

On board:
- Apache
- MariaDB
- PHP 8.1.11-fpm with xDebug

PHP 8.1.11-fpm built in docker/Dockerfile and contains:
- GD --with-freetype --with-jpeg --with-webp
- mysqli
- xDebug
- Composer

First run:
- docker compose up

or:

- docker-compose up

Next usage:

 docker compose start\stop

or...
